# Origin — Brand Guide

## Name

**Origin**

## Why "Origin"

Code editors improved how you write code. Origin makes writing code irrelevant. The name reflects this:

- **Origin of your product** — every product starts here
- **Origin of a new era** — the shift from writing code to thinking in systems
- **Origin story** — every builder has one, this is where it begins
- **The origin point (0,0)** — the mathematical point from which everything is measured

## Positioning

Origin is not a code editor. It is not an IDE. It is not a better Cursor. It is the thing that comes after all of them. Writing code is sending pigeons with letters. Cursor made the pigeons faster. Origin is email.

## Tagline

**"Start here."**

## Logo Concept

A single dot. The origin point. Everything expands outward from it.

## Voice

- Confident, not arrogant
- Simple, not dumbed down
- For creators, not engineers
- Anti-complexity, pro-clarity

## Key Messaging

- "Before Origin, you needed to be an engineer. After Origin, you just needed an idea."
- "Cursor improved the way you write code. Origin made writing code irrelevant."
- "Where does your product start? Origin."
